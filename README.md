# ORL: Novel Sampling Strategy for Inference in M/M/1 Queues

This repository contains the implementation and computational experiments related to the article:

**E.S. Gomes, F.R.B. Cruz**  
*Novel sampling strategy for inference in M/M/1 queues*  
Operations Research Letters, Volume 67, 2026, 107461  
ISSN: 0167-6377  

DOI: https://doi.org/10.1016/j.orl.2026.107461

## Abstract

We address the estimation of performance measures in single-server Markovian queueing systems by introducing a novel sampling scheme based on the number of departures that occur during the interarrival time of a customer. This approach enables sequential sample collection from a stochastic process with strong dependence structure, leading to improved efficiency. The proposed model is applied to both classical and Bayesian inference of key queue performance measures.

## Repository Structure

- `experiments.R`  
  Contains the code required to reproduce all numerical experiments presented in the article.

- `plots.R`  
  Reproduces all figures presented in the article.

- `results/`  
  Stores numerical outputs generated by the experiments.

- `figures/`  
  Stores generated plots and graphical outputs.

## Reproducibility

To reproduce the results reported in the paper:

1. Run:

```r
source("experiments.R")
```

2. Generate figures with:

```r
source("plots.R")
```
